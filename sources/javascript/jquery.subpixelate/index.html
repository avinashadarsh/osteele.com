JQuery Subpixelate Plugin

This plugin replaces text in a page by glyphs from miha's subpixel microfont. It's not practical, portable, efficient, or well-written (writing the plugin was an after-dinner project), but it's fun.

Specifically, the plugin defines a function that replaces each letter in the text of an element by a div whose background image extracts a rectangle from a font sampler image. By default, it hot-links to the image on Typophile. This worked on the day I wrote it.

The plugin itself is available under the MIT License. The image file that it links to is copyright by miha. Generated images are likewise the property of miha, and are released under this license: CC by-nd 2.5. The latter license will be changed once miha finishes the font.

Usage

<div class="substitute">
  His followers called him Mahasamatman, and said he was a god.
</div>

<script type="text/javascript">
  $(function() {
    $('.substitute').subpixelate();
  });
</script>
