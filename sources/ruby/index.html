OpenLaszlo Ruby Code

ROpenLaszlo Gem

The ROpenLaszlo Ruby gem (formerly the "OpenLaszlo Ruby library") provides an API or compiling OpenLaszlo programs from Ruby, for use in Rake tasks and Rails integration.

ROpenLaszlo is now hosted on RubyForge.

Contacts Demo

This is a version of the OpenLaszlo "contact list" example, modified to work with Rails. It will be used as an extraction source for a set of Ruby on Rails helper functions.
