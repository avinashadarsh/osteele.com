OpenLaszlo Ruby Code

ROpenLaszlo Gem

The ROpenLaszlo Ruby gem (formerly the "OpenLaszlo Ruby library") provides an API for compiling OpenLaszlo programs from Ruby, for use in Rake tasks and Rails integration.

ROpenLaszlo is now hosted on RubyForge.

Installation: gem install ropenlaszlo

OpenLaszlo Ruby Plugin

This plugin provides scaffolding and helpers that implement the OpenLaszlo REST API, making it easy to integrate OpenLaszlo with Rails.

Installation:

script/plugin install svn://rubyforge.org/var/svn/ropenlaszlo/rails/plugins/openlaszlo

Contacts Demo

This is a version of the OpenLaszlo "contact list" example, modified to work with Rails. It uses the ROpenLaszlo gem and the OpenLaszlo plugin above.
