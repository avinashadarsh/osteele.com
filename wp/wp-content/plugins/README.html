flickrRSS Plugin Readme

This plugin allows you to display a flickr style badge (for a tag, user or group pool) on your WordPress site. It uses MagpieRSS to parse the RSS feed and display the images. This is my first attempt at a WordPress plugin, so you may run into some problems I hadn't considered. If you use the image cache, you'll need to remove old images manually.

Features

  • Display a badge for any tag, userid, user tag or group pool photo stream
  • Configuration panel
  • Displays image titles
  • Choose small, thumbnail medium sized images
  • Can cache images locally
  • Generates valid code

Manual Installation

  1. Unpack the files
  2. Upload flickrrss.php to your plugins directory.
  3. Put the cache dir somewhere.
  4. Make the cache dir writable.
  5. Activate the plugin.
  6. Go to the configuration panel in Options.
